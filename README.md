# MiraiHardware — Site vitrine

Site vitrine pour une activité de **montage de PC sur mesure, réparation et conseil informatique**.

Site one-page moderne, responsive, sombre/tech, en HTML / CSS / JavaScript pur — sans framework ni dépendance lourde.

## Contenu

| Fichier | Rôle |
|---|---|
| `index.html` | Page principale (hero, prestations, configs, méthode, avis, contact) |
| `mentions.html` | Mentions légales |
| `style.css` | Toute la feuille de style |
| `script.js` | Interactions : menu mobile, animations au scroll, compteurs, formulaire |

## Lancer en local

Aucune installation nécessaire : ouvrir `index.html` dans un navigateur.

## Mise en ligne (GitHub Pages)

1. Pousser les fichiers sur la branche `main`
2. Repo → **Settings** → **Pages**
3. Source : branche `main`, dossier `/ (root)`
4. Le site est publié sur `https://billalbzn.github.io/MiraiHardware/`

## À personnaliser

- L'adresse email `contact@miraihardware.fr` (dans `index.html`, `script.js`, `mentions.html`)
- Les prix et configurations dans la section « Configurations »
- Les informations légales dans `mentions.html`
- Les avis clients
